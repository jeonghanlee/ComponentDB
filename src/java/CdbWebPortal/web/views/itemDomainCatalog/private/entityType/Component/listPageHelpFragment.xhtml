Component:

  • By definition, it is an abstract part or a constituent element of a larger whole (e.g., of a mechanical, electrical, or software system)
  • Represents completed and un-modifiable product or an assembled product that can be found in the Component Catalog
  • Any modification would be considered a new component (e.g., with a new revision or model number)

Assembly (Complex) Component:

  • It is a component that is made of other components found in the Component Catalog
  • It is not a “work in progress” or something that can be customized; it can only be obtained “as-is” from the Component Catalog
  • Assembly component instance may be associated with child component instances
