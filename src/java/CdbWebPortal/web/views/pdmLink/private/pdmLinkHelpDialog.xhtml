When a full drawing number is entered, no search results will appear. Only the information for the particular drawing.

  • ex: D14100201-113000.ASM
When partial drawing number, keywords, or wildcards are used. Search results will appear.
  • Using Wildcards
    • * - Fill in any amount of characters
      • ex: D14100201-113*.ASM
    • ? - Fill in exactly one character
      • ex: D14100201-113?1?.???
  • Using Keywords
    • Specify keywords that describe the drawing
      • ex: dipole support assembly drw
Clicking on a search result will load drawing details.
